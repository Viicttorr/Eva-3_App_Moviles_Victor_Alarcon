# Aplicación Móvil IoT - Evaluación Sumativa N°3

Este proyecto consiste en una aplicación móvil desarrollada en Android Studio para la asignatura de Aplicaciones Móviles para IoT (TI3042) en INACAP. La aplicación ha evolucionado desde una base funcional de autenticación (Evaluación 2) hacia una plataforma conectada a servicios en la nube utilizando Firebase.
